# HTML/CSS 마스터 체크리스트 - 실무 중심 웹 UI 설계 및 구현

이 문서는 HTML과 CSS만을 활용하여 실제 웹사이트에서 사용되는 모든 UI 구성 요소를 학습하고 구현하기 위한 종합적인 체크리스트입니다. 15개 이상의 웹사이트 장르와 100여 개의 핵심 컴포넌트를 다루며, 단계별 학습과 실무 적용을 목표로 합니다.
